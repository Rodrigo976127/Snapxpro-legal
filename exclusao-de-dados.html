Exclusão de Dados

O SnapX Pro não armazena dados pessoais em servidores externos. Todas as informações e imagens permanecem no próprio dispositivo do usuário.

Para excluir permanentemente todos os dados do aplicativo, basta desinstalar o app do seu dispositivo.

Se você estiver usando uma conta vinculada (como o login com Google), poderá solicitar a remoção de informações relacionadas entrando em contato pelo e-mail: rdxstudio.suporte@gmail.com.
